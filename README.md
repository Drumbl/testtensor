# Goods and Services

Это тестовое задание под Android. Можно использовать как пример работы с некоторыми сторонними библиотеками. Приложение запрашивает категории товаров и услуг в формате JSON, сохраняет в SQLite и выводит на экран в виде дерева.

### Используемые библиотеки

* [Retrofit 2](http://square.github.io/retrofit/)
* [AndroidTreeView](https://github.com/bmelnychuk/AndroidTreeView)
